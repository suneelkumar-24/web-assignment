📚 Step 1: Creating a Database
💡 What is a Database?
A database is an organized collection of structured data. Think of it as a digital filing cabinet where you can store, manage, and retrieve information efficiently.

Basic Syntax:

CREATE DATABASE database_name;

Example - Creating a Student Database:

CREATE DATABASE student_management_system;

Using the Database:

USE student_management_system;
⚠️ Important: Always use the USE command before creating tables or inserting data into your database!
📊 Step 2: Creating Tables
💡 What is a Table?
A table stores data in rows and columns, similar to an Excel spreadsheet. Each column represents a specific attribute, and each row represents a record.

Basic Syntax:

CREATE TABLE table_name (
    column1 datatype constraint,
    column2 datatype constraint,
    column3 datatype constraint
);

Example - Creating a Students Table:

CREATE TABLE students (
    student_id INT PRIMARY KEY AUTO_INCREMENT,
    first_name VARCHAR(50) NOT NULL,
    last_name VARCHAR(50) NOT NULL,
    email VARCHAR(100) UNIQUE,
    age INT,
    grade CHAR(2),
    city VARCHAR(50),
    enrollment_date DATE
);

Common Data Types:

INT          - Integer numbers
VARCHAR(n)   - Variable text (max n characters)
DATE         - Date values (YYYY-MM-DD)
DECIMAL(p,s) - Decimal numbers
BOOLEAN      - True/False values
TEXT         - Long text

🔍 Step 4: SELECT and DISTINCT Queries

SELECT Statement - Retrieve Data

-- Select all columns
SELECT * FROM students;

-- Select specific columns
SELECT first_name, last_name, email FROM students;

-- Select with conditions
SELECT * FROM students WHERE age > 21;

-- Select with ordering
SELECT * FROM students ORDER BY last_name ASC;

DISTINCT Statement - Remove Duplicates

-- Get unique cities
SELECT DISTINCT city FROM students;

-- Get unique grade values
SELECT DISTINCT grade FROM students;

-- Get unique age values
SELECT DISTINCT age FROM students ORDER BY age;
